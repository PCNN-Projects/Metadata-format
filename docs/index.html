Post-COVID Metadata Template

This structured template can be used to collect metadata from post-COVID cohort studies and has been used to do so for Dutch cohort studies as part of the Post-COVID Network Netherlands (PCNN). The metadata file includes key information on study design, participants, outcomes, and methods of measurement. The goal is to create an online metadata dictionary, providing valuable insights into the long-term effects of COVID-19.
